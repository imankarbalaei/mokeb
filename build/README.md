# آیکن فایل exe

برای تنظیم آیکن خروجی ویندوز، فایل آیکن را با این نام و مسیر قرار بدهید:

```text
build/icon.ico
```

نکات:

- فرمت باید `.ico` باشد، نه `.png` یا `.jpg`.
- بهتر است فایل ico شامل اندازه‌های 16، 32، 48، 128 و 256 پیکسل باشد.
- بعد از قرار دادن فایل، دوباره خروجی بگیرید:

```bash
npm run dist:win32
```

اگر سیستم مقصد ۶۴ بیتی است:

```bash
npm run dist:win64
```
