# لوگوی داخل برنامه

برای نمایش لوگو کنار عنوان «موکب الشهدا قم»، فایل PNG را با این نام و مسیر قرار بدهید:

```text
public/logo.png
```

بعد از قرار دادن فایل، برنامه را دوباره اجرا یا خروجی exe را دوباره بسازید:

```bash
npm run dist:win32
```

نکته:

- فرمت باید PNG باشد.
- نام فایل دقیقاً باید `logo.png` باشد.
- اگر فایل وجود نداشته باشد، برنامه بدون لوگو نمایش داده می‌شود و خطا نمی‌دهد.
